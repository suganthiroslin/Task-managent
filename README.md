# 📝 Task Management System

A simple web-based Task Management System built using **Flask (Python)** and **Oracle Database**.  
This application allows users to add, view, update, and delete tasks with due dates and status tracking.

---

## 🚀 Features

- ➕ Add new tasks with due date
- 📋 View all tasks in a clean list format
- ✅ Update task status (Completed / Yet to Complete / Not Done)
- 🗑 Delete tasks
- 🎨 Modern Glassmorphism UI
- 🔗 REST API integration with Oracle Database

---

## 🛠 Tech Stack

- **Frontend**: HTML, CSS (Glassmorphism), JavaScript
- **Backend**: Flask (Python)
- **Database**: Oracle Database
- **Driver**: oracledb (Thin Mode)
